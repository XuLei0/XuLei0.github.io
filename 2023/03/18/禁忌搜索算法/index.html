禁忌搜索（Tabu Search）算法

TS主要构成要素

  1. 领域：简单地说就是给定点附近其他点的集合。领域就是指对当前解进行一个操作（这个操作称之为领域动作）可以得到的所有解的集合。
  2. 领域动作：领域动作是一个函数，通过这个函数，对当前解s，产生其相应的领居解集合。
  3. 禁忌表：
    1. 禁忌对象：由于需要避免一些操作的重复进行，就要将一些元素放到禁忌表中以禁止对这些元素进行操作，这些元素就是我们指的紧急对象（通常指找到的局部最优解）。
    2. 禁忌长度：禁忌长度是被禁对象不允许选取的迭代次数。一般是给被禁对象 x 一个数（禁忌长度）t，要求对象 x 在 t 步迭代内被禁止
  4. 候选集合：候选集合由邻域中的邻居组成，常规方法是从邻域中选择若干个目标函数取值最佳的加入集合
  5. 评价函数：评价候选集合元素选取的一个评价公式。
  6. 特赦规则：在禁忌搜索算法迭代过程中，会出现候选集中全部对象都被禁忌，或有一对象被禁，但若解禁则其目标值将有非常大的下降情况。这种情况为了达到全局最优，我们会让一些禁忌对象重新可选。这种方法称为特赦，相应的规则称为特赦规则。

TS算法流程

  1. 禁忌表 H = 空集，并选定一个初始解 $x_{1}$ ;
  2. 在 $x_{i}$ 的邻域 $N(x_{i})$ 中选择候选集 $Can_N(x_{i})$ ；在 $Can_N(x_{i})$ 中选择一个评价值最佳的解 $x_{i+1}$ ，这个 $x_{i+1}$ 不会和 $x_{i}$ 进行比较，这就有助于跳出局部最优解。
    • $x_{i+1}$ 不在禁忌表中，选为当前解。
    • $x_{i+1}$ 在禁忌表中，满足破禁条件（即优于当前搜索中的最优解），$x_{i+1}$ 为当前解，否则选择一个不在禁忌表中的次优解。
  3. 重复步骤 2 ，直到满足条件为止（一般终止条件为是否达到一定的迭代次数或者达到了一个时间限制）。

禁忌搜索求解TSP问题

假设禁忌长度设为 2，候选集合长度定义为 4，迭代次数为 100

给定初始解 $x_{1} = 【A,D,C,F,E,A】, f(x_{1})=10$，历史最优为 10

候选集和 禁忌表
【A,C,D,F,E,A】f=15
【A,D,C,E,F,A】f =20
【A,D,F,C,E,A】f=8
【A,E,C,F,D,A】f=6

我们发现对 $x_{1}$ 交换 D 和 E 时，f 最优，此时 $x_{2} = 【A,E,C,F,D,A】, f(x_{2}=6)$, 历史最优为 6， 将 DE 放入禁忌表。

候选集合 禁忌表
【A,E,F,C,D,A】f=9 D-E
【A,F,C,E,D,A】f=15
【A,C,E,F,D,A】f=6
【A,E,D,F,C,A】f=5

对 $x_{2}$ 交换 D 和 C 时，f 最优，此时 $x_{3} = 【A,E,D,F,C,A】,f(x_{3})=5$ , 历史最优为 5，将 DC 加入到禁忌表中。

候选集合 禁忌表
【A,E,C,F,D,A】f=8 D-E
【A,E,F,D,C,A】f=10 D-C
【A,E,D,C,F,A】f=14
【A,C,D,F,E,A】 f=16

此时，最优的是将 $x_{3}$ 中的 D 和 C 交换，但由于D-C 在禁忌表中，因此我们退而求其次，对 $x_{3}$ 交换 F 和 D, 此时得到 $x_{4}=【A,E,F,D,C,A】,f(x_{4}) = 10$ ，历史最优为 5，将 F-D 放入禁忌表，由于禁忌表长度为 2，所以要将 D-E 移出禁忌表，再将 F-D放入。

候选集合 禁忌表
【A,E,F,C,D,A】f=4 D-C
【A,E,C,D,F,A】f=5 F-D
【A,F,E,D,C,A】f=7
【A,C,F,D,E,A】 f=10

这里 $x_{4}$ 交换 C-D，得到了 $x_{5}$，这里 C-D 是在禁忌表中的，但由于 $f(x_{5})=4$ 比当前历史最优的还要小，满足特赦规则。然后再将 C-D 重新加入禁忌表中。

候选集合 禁忌表
【A,C,F,E,D,A】f=5 D-F
【A,E,C,F,D,A】f=7 C-D
【A,D,F,C,E,A】f=9
【A,F,E,C,D,A】 f=29

依次迭代，直到达到停止条件。

禁忌搜索解TSP代码

# -- coding: utf-8 --
import random
import math
from collections import deque       # 用来模拟禁忌表


def get_cost(n, adj_mat, sol):
    return sum([adj_mat[sol[_]][sol[(_+1) % n]] for _ in range(n)])


def tnm_selection(n, adj_mat, sol, max_tnm, mut_md, tb_size, tb_list, fq_dict, best_cost):
    """
    :param n: 向量的数量，代表城市的个数
    :param adj_mat: 邻接矩阵，用来记录每个城市之间的距离
    :param sol: 当前解
    :param max_tnm: 候选集合的个数
    :param mut_md: [get_sol, get_delta] 突变的方法，例如，交换、2-opt
    :param tb_size: 禁忌表的长度
    :param tb_list: deque,禁忌表
    :param fq_dict: 顶点对的访问次数
    :param best_cost: 记录历史最优的解
    :return:
    """
    get_new_sol = mut_md[0]
    get_delta = mut_md[1]
    cost = get_cost(n, adj_mat, sol)    # 求一个解的目标函数

    # delta 的作用是用于评估邻居解的质量和选择最优解
    best_delta_0 = math.inf
    best_i_0 = best_j_0 = -1

    best_delta_1 = math.inf
    best_i_1 = best_j_1 = -1

    for _ in range(max_tnm):
        i, j = random.sample(range(n), 2)        # 随机选择两个索引下标
        i, j = (i, j) if i < j else (j, i)       # 让 i < j
        v_1, v_2 = (sol[i], sol[j]) if sol[i] < sol[j] else (sol[j], sol[i])

        delta = get_delta(n, adj_mat, sol, i, j)
        if (v_1, v_2) not in tb_list:      # 不违反禁忌规则的 delta 最小值
            if delta < best_delta_0:
                best_delta_0 = delta
                best_i_0 = i
                best_j_0 = j
        else:                              # 违反禁忌规则的 delta 最小值
            if delta < best_delta_1:
                best_delta_1 = delta
                best_i_1 = i
                best_j_1 = j

    # 找到最小的 best_delta_0 或者 best_delta_1
    if best_delta_1 < best_delta_0 and cost + best_delta_1 < best_cost:     # 满足特赦规则
        v_1, v_2 = (sol[best_i_1], sol[best_j_1]) if sol[best_i_1] < sol[best_j_1] else (sol[best_j_1], sol[best_i_1])
        tb_list.remove((v_1, v_2))
        tb_list.append((v_1, v_2))
        fq_dict[(v_1, v_2)] = fq_dict.get((v_1, v_2), 0) + 1  # 如果没有就赋值为 0 再加 1
        new_sol = get_new_sol(sol, best_i_1, best_j_1)
        new_cost = cost + best_delta_1
    else:
        if tb_size > 0:
            v1, v2 = (sol[best_j_0], sol[best_j_0]) if (sol[best_i_0] < sol[best_j_0]) else (sol[best_j_0], sol[best_j_0])
            if len(tb_list) == tb_size:
                tb_list.popleft()
            tb_list.append((v_1, v_2))
            fq_dict[(v_1, v_2)] = fq_dict.get((v_1, v_2), 0) + 1
        new_sol = get_new_sol(sol, best_i_0, best_j_0)
        new_cost = cost + best_delta_0
    return new_sol, new_cost, tb_list, fq_dict


def tb(n, adj_mat, tb_size, max_tnm, mut_md, term_count):
    """
    :param n:
    :param adj_mat:
    :param tb_size:
    :param max_tnm:
    :param mut_md:
    :param term_count: 经过 term_count 迭代后就退出
    :return:
    """
    sol = list(range(n))
    random.shuffle(sol)       # 打乱
    tb_list = deque([])
    fq_dict = {}
    best_sol = sol.copy()
    best_cost = get_cost(n, adj_mat, sol)
    data = {'cost':deque([]), 'best_cost':deque([])}
    count = 0
    while True:
        sol, cost, tb_list, fq_dict = tnm_selection(n, adj_mat, sol, max_tnm, mut_md, tb_size, tb_list, fq_dict, best_cost)
        if cost < best_cost:
            best_sol = sol
            best_cost = cost
            count = 0
        else:
            count += 1
        data['cost'].append(cost)
        data['best_cost'].append(best_cost)
        if count > term_count:
            break
    data['fq_dict'] = fq_dict
    return best_sol, best_cost, data
