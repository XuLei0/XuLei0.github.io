蚁群优化算法

1. 仿生物学原理

蚂蚁在寻找食物源时，会在其经过的路径上释放一种信息素，并能够感知其它蚂蚁释放的信息素。信息素浓度的大小表征到食物源路径的远近，信息素浓度越高，表示对应的路径距离越短。通常，蚂蚁会以较大的概率优先选择信息素浓度较高的路径，并释放一定量的信息素，以增强该条路径上的信息素浓度，这样会形成一个正反馈。最终，蚂蚁能够找到一条从巢穴到食物源的最佳路径，即最短距离。值得一提的是，生物学家同时发现，路径上的信息素浓度会随着时间的推进而逐渐衰减。

启发： 用蚂蚁的行走路径表示待优化问题的可行解，整个蚂蚁群体的所有路径构成待优化问题的解空间。路径较短的蚂蚁释放的信息素量较多，随着时间的推进，较短的路径上积累的信息素浓度逐渐增高，选择该路径上的蚂蚁个数也越来越多。最终，整个蚂蚁会在正反馈的作用下集中到最佳的路径上，此时对应的便是待优化问题的最优解。

  • 蚁群算法的优势

    （1）易获得全局最优解；

    （2）应用面广，易与其他问题结合；

    （3）分布式计算方式，多个个体并行运算，大大提升算法的运行效率；

    （4）鲁棒性强，采用正反馈机制，使得搜索过程不断收敛，逼近最优解，通常应用于图论算法；

  • 蚁群算法劣势

​ （1）虽然一定会收敛，但无法确定收敛所需的时间；

​ （2）理论分析较为困难，算法更侧重于实践层面；

2. 蚁群算法解决TSP问题

  • 算法原理：

​ 蚂蚁群体数量为 $m$ , 城市数量 $n$, 城市 $i$ 和城市 $j$ 之间的距离为 $d_{ij}$，$t$ 时刻城市 $i$ 与 城市 $j$ 路上的信息素浓度为 $\tau_{ij}(t)$。初始时刻，各个城市间连接路径上的信息素浓度相同，不妨设 $\tau_{ij}(0) = \tau_{0}$ 。

蚂蚁 $k,(k=1,2,…m)$ 根据各个城市间连接路径上的信息素浓度决定下一个访问城市，设 $P_{ij}^{k}(t)$ 表示 $t$ 时刻蚂蚁 $k$ 从城市 $i$ 转移到 城市 $j$ 的转移概率，其公式为：

其中 $\eta_{ij}(t) = \frac{1}{d_{ij}}$ 为启发式函数，表示蚂蚁从城市 $i$ 转移到城市 $j$ 的期望程度，路径越长越不期望走这条路；$allow_{k}$ 为蚂蚁 $k$ 待访问城市的集合；$\alpha$ 为信息素重要程度因子，其值越大，表示信息素的浓度在转移过程中起的作用越大；$\beta$ 为启发式函数重要程度因子，其值越大，表示启发式函数在转移过程中作用越大，即蚂蚁会以较大的概率转移到距离短的城市。如果 $\alpha$ 为 0，那么蚂蚁转移不依据信息素，即群体经验，算法就转变为贪心算法。如果 $\beta = 0$，那就不会依据距离来转移，完全根据群体的经验。经过 $n$ 个时刻，所有蚂蚁走完了所有的城市，完成了一次循环，每只蚂蚁所走过的路径就是一个解。

由于蚂蚁释放信息素的同时，各个城市间路径上的信息素会逐渐消失，设参数 $\rho (0 < \rho < 1)$ 表示信息素的挥发程度。因此，当所有蚂蚁完成一次循环后，各个城市间信息素浓度需要进行实时更新，即：

$\bigtriangleup\tau_{ij}^{k}$ 表示第 $k$ 只蚂蚁在城市 $i$ 与 城市 $j$ 连接路径上释放的信息素浓度；$\bigtriangleup\tau_{ij}$ 表示所有蚂蚁在城市 $i$ 和城市 $j$ 连接路径上释放的信息素浓度之和。

蚂蚁释放信息素的模型：

（1）Ant Cycle System模型

​ $\bigtriangleup\tau_{ij}^{k} = \frac{Q}{L_{k}}$，第 $k$ 只蚂蚁从城市 $i$ 访问城市 $j$ , 其中 $Q$ 表示一只蚂蚁一个循环所能产生的信息素总量；$L_{k}$ 为第 $k$ 只蚂蚁经过路径的长度。

（2）Ant Quantity System

​ $\bigtriangleup\tau_{ij}^{k} = \frac{Q}{d_{ij}}$ , 第 $k$ 只蚂蚁从城市 $i$ 访问城市 $j$

（3）Ant Density System

​ $\bigtriangleup\tau_{ij}^{k} = Q$

​ 一般选用 Ant Cycle System 模型来计算释放信息素的浓度，蚂蚁经过的路径越短，释放信息素的浓度越高。

  • 算法步骤

    （1）初始化参数

    ​ 需要对相关参数进行初始化，入蚁群规模（蚂蚁数量）m、信息素重要程度因子 $\alpha$ 、启发函数重要程度因子 $\beta$ 、信息素挥发程度因子 $\rho$ 、信息素释放总量 $Q$、最大迭代次数 $iter_{max}$ 。

    ​ 同时，需要计算一个距离矩阵，即两两城市之间的相互距离，从而得到对称的距离。由于启发函数为 $\eta_{ij}(t) = \frac{1}{d_{ij}}$ ，为了保证分母不为 0 , 需要将对角线上的元素修正为一个比较小的正数（$10^{-4}或10^{-5}$）

    （2）构建解空间

    ​ 将各个蚂蚁随机的放置于不同的出发点，对每个蚂蚁 $k$ ，按照转移概率的计算公式，确定其下一个待访问的城市（轮盘赌法选择下一步的城市），直到所有蚂蚁访问完所有城市，即构造完一组路径。

    （3）更新信息素

    ​ 计算各个蚂蚁经过的路径长度 $L_{k}$ , 根据信息素迭代公式对各个城市路径上的信息素浓度进行更新。同时，记录当前迭代次数中的最优解（最短路径）。

    （4）判断是否终止，若达到最大的迭代次数则终止计算，输出最优解，否则，清空蚂蚁经过路径的记录表，并返回步骤 2。

    aco.py 

    # -- coding: utf-8 --
import random


class Graph(object):
    """
    用来存储整个图中所有城市之间的距离以及信息素
    """
    def __init__(self, cost_matrix, rank):
        self.matrix = cost_matrix
        self.rank = rank
        self.pheromone = [[1/(rank * rank) for j in range(rank)] for i in range(rank)]     # 初始化信息素为 1/(num_city * num_city)


class ACO(object):
    def __init__(self, ant_count, generations, alpha, beta, rho, q, strategy):
        """
        :param ant_count: 蚁群数量
        :param generations: 迭代次数
        :param alpha: 信息素的重要程度
        :param beta:  启发式信息重要程度
        :param rho:   信息素挥发率
        :param q:     一个个体一次循环所拥有的信息素的量
        :param strategy:   信息素更新策略
        """
        self.Q = q
        self.rho = rho
        self.beta = beta
        self.alpha = alpha
        self.ant_count = ant_count
        self.generations = generations
        self.update_strategy = strategy


    # 更新信息素
    def _update_pheromone(self, graph, ants):               # ants 是所有蚂蚁所表示的解，即走过的路径
        for i, row in enumerate(graph.pheromone):
            for j, col in enumerate(graph.pheromone):
                graph.pheromone[i][j] *= self.rho           # 首先，当前已有的信息素是会挥发的
                for ant in ants:
                    graph.pheromone[i][j] += ant.pheromone_delta[i][j]      # 其次要加上这些蚂蚁走过 i-j 这条边时留下的信息素，这个蚂蚁没有走过的边信息素为 0

    def solve(self, graph):
        # 记录蚁群中最好的解
        best_cost = float('inf')
        best_solution = []
        for gen in range(self.generations):
            ants = [_Ant(self, graph) for i in range(self.ant_count)]       # 这里的 self 是把 ACO 这个类传进 _Ant 类的 __init__() 方法
            for ant in ants:
                # 对每个蚂蚁都要进行 rank - 1 次选择下一次的结点
                for i in range(graph.rank - 1):
                    ant._select_next()
                ant.total_cost += graph.matrix[ant.tabu[-1]][ant.tabu[0]]
                # 更新最优解
                if ant.total_cost < best_cost:
                    best_cost = ant.total_cost
                    best_solution = [] + ant.tabu
                ant._update_pheromone_delta()    # 每个蚂蚁产生新的信息素
            # 更新图中整体的信息素
            self._update_pheromone(graph, ants)
        return best_solution, best_cost



class _Ant(object):
    def __init__(self, aco:ACO, graph):
        self.colony = aco
        self.graph = graph
        # 记录解的结果
        self.total_cost = 0.0
        self.tabu = []               # tabu_list, 可以用来记录所走的边
        self.pheromone_delta = []    # 该蚂蚁在每条边上产生的信息素的量
        self.allowed = [i for i in range(graph.rank)]
        self.eta = [[0 if i==j else 1/graph.matrix[i][j] for j in range(graph.rank)] for i in range(graph.rank)]     # 启发式信息
        start = random.randint(0, graph.rank-1)     # 开始从任意一个城市
        self.tabu.append(start)          # 这个点被访问过了就不会被访问了
        self.current = start
        self.allowed.remove(start)       # allow 集合

    def _select_next(self):
        denominator = 0
        # 先计算分母
        for i in self.allowed:
            denominator += self.graph.pheromone[self.current][i] ** self.colony.alpha * self.eta[self.current][i] ** self.colony.beta
        # 转移概率
        probabilities = [0 for i in range(self.graph.rank)]
        for i in range(self.graph):
            if i not in self.allowed:
                continue
            probabilities[i] = self.graph.pheromone[self.current][i] ** self.colony.alpha * \
                self.eta[self.current][i] ** self.colony.beta / denominator

        # 通过轮盘赌法来选择下一步选择的结点
        selected = 0
        rand = random.random()
        for i, probabilities in enumerate(probabilities):
            rand -= probabilities
            if rand <= 0:
                selected = i
                break

        self.allowed.remove(selected)
        self.tabu.append(selected)
        self.total_cost += self.graph.matrix[self.current][selected]
        self.current = selected

    # 计算这个蚂蚁释放的信息素的浓度
    def _update_pheromone_delta(self):
        self.pheromone_delta = [[0 for j in range(self.graph.rank)] for i in range(self.graph.rank)]
        for _ in range(1, len(self.tabu)):
            i = self.tabu[_ - 1]
            j = self.tabu[_]
            if self.colony.update_strategy == 1:       # ant-density system
                self.pheromone_delta[i][j] = self.colony.Q
            elif self.colony.update_strategy == 2:     # ant-quantity system
                self.pheromone_delta[i][j] = self.colony.Q / self.graph.matrix[i][j]
            else:
                self.pheromone_delta[i][j] = self.colony.Q / self.total_cost

    main.py

    # -- coding: utf-8 --
import math
from aco import ACO, Graph
from plot import plot


def distance(city1, city2):
    return math.sqrt((city1['x'] - city2['x'])**2 + (city1['y'] - city2['y'])**2)


def main():
    cities = []
    points = []
    with open('./data/chn31.txt') as f:
        for line in f.readlines():
            city = line.split(' ')
            cities.append(dict(index=int(city[0]), x=int(city[1]), y=int(city[2])))     # 存放城市字典
            points.append((int(city[1]), int(city[2])))                                 # 城市位置

    cost_matrix = []
    rank = len(cities)
    for i in range(rank):
        row = []
        for j in range(rank):
            row.append(distance(cities[i], cities[j]))
        cost_matrix.append(row)
    aco = ACO(10, 100, 1.0, 10.0, 0.5, 10, 2)
    graph = Graph(cost_matrix, rank)
    path, cost = aco.solve(graph)
    print('cost:{}, path:{}'.format(cost, path))
    plot(points, path)


if __name__ == "__main__":
    main()

    plot.py

    # -- coding: utf-8 --
import operator
import matplotlib.pyplot as plt


def plot(points, path):
    x = []
    y = []
    for point in points:
        x.append(point[0])
        y.append(point[1])
    y = list(map(operator.sub, [max(y) for i in range(len(points))], y))  # 将所有点的 y 轴坐标修改为与最高点的距离
    plt.plot(x, y, 'co')

    for _ in range(1, len(path)):
        i = path[_ - 1]
        j = path[_]
        plt.arrow(x[i], y[i], x[j]-x[i], y[j]-y[i], color='r', length_includes_head=True)  # 前两个坐标为箭头尾坐标，后两个是箭头的长度和方向

    plt.xlim(0, max(x)*1.1)
    plt.ylim(0, max(y)*1.1)
    plt.show()
