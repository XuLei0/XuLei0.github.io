模拟退火算法（Simulated Annealing）

  1. 求一个给定函数的最值问题

$y = 11sinx + 7cos(5x)$ ，在$[-3,3]$内的最大值，暴力搜索，如果某个函数有30个变量呢？暴力搜索的时间复杂度就会很高

如果要求最小值，那么久将函数添加负号即可转换为求最小值问题

  1. 蒙特卡洛模拟（这里用其求解最值问题）

​ 思想：随机生成很多组解，验证这些解是否满足条件，若满足则将其保存到一个 “可行集” 中，然后计算这个可行集内每个解对应的目标函数值，在里面找到最值即可。

和穷举法的区别：穷举法是按一定规则来穷举整个空间，蒙特卡洛模拟是随机搜索，这是两者的区别。

  1. 启发式搜索简单总结为下面两点：
    • 任何有助于找到问题的最优解，但不能保证找到最优解的方法均是启发式方法
    • 有助于加速求解过程和找到较最优解的方法是启发式算法
  2. 爬山法

​

​ 爬山法怎么找到最大值，向周围的方向走，哪个方向会得到更大的目标函数值就往哪个方向走，直到找到一个极大值点，此时结束搜索。爬山法的缺点是很容易找到局部最优解，关键是初始值的选取，模拟退火可以解决这个问题。

​ 爬山法也是属于启发式算法。

  1. 模拟退火算法

模拟退火会以一定的概率接收一个比当前解要差的解，此时就有几率跳出局部最优解，从而找到全局最优解。

例子：求解 $y = 11sinx + 7cos(5x)$ ,在 $[-3,3]$ 内的最大值

  • 假设当前搜索到了 $x_{i}$ , 对应的函数值为 $f(x_{i})$ , 在 $x_{i}$ 附近随机找一个新的解 $x_{j}$ ，对应的函数值为 $f(x_{j})$

    • 如果 $f(x_{j}) > f(x_{i})$ , 则接受这个新解 $x_{j}$

    • 如果 $f(x_{j}) \le f(x_{i})$ , 以一定的概率 $p \in[0,1]$ 接受这个解，而且 $f(x_{j})$ 越接近 $f(x_{i})$ , $p$ 越大（这种想法很直观，新解和旧解的函数值越接近，那么就更愿意接受这个新解），即 $p$ 正比于 $\frac{1}{|f(x_{i})-f(x_{j})|}$ , 当 $p$ 为 0 时，等于爬山法，等于 1 时，等于蒙特卡洛模拟。

  • $y = e^{-x}, x\ge 0$ 这个函数的值域在 $[0,1]$ 之间， 用该函数当作概率函数，且该函数是递减的，那么可以假设 $p \propto e^{-|f(x_{i}) - f(x_{j})|}$

  • 接受新解的概率 $p$ 越大，意味着我们更容易接受不好的解，因此解空间的搜索范围越大。

  • 假设搜索过程需要经过很多次迭代，在搜索前期我们的搜索范围应该尽量大，搜索后期尽量小。

  • 因此我们概率函数 $p \propto e^{-|f(x_{i})-f(x_{j})|\times C_{t}}$，其中 $C_{t}$ 可以看作一个和时间有关的系数，因此在搜索过程中，我们接受新解的概率 $p$ 就和时间有关，显然时间越长，$C_{t}$ 越 大。

  • 问题：

    • 如果优化问题有约束条件怎么办？
      • 在生成新解时，判断是否满足约束条件，不满足则重新生成。
    • 这个 $C_{t}$ 怎么设置？
      • $T_{t} = \alpha^{t}T_{0} = 100\times0.95^{t}$ ，100是初始温度，0.95是冷却系数，这都是可调参数，$T_{t}$ 随着时间增长而减小。
      • $C_{t} = \frac{1}{T_{t}} = \frac{1}{100\times0.95^{t}}$ ，那么 $p_{t} = e^{-\frac{|f(x_{i})-f(x_{j})|}{100\times0.95^{t}}}$
    • 什么时候停止搜索？
      • 达到指定的迭代次数
      • 达到指定的温度，例如温度小于 0.000001，温度很低时，基本上产生新解的概率很低了
      • 连续找到最优解 M 次，例如连续找到了 30 次
    • 怎么生成一个新解？
      • 没有统一规定，具体问题具体分析，但是要根据当前解生成一个新解，而不是随机的

  1. 模拟退火解TSP问题

    # -- coding: utf-8 --
import math
import random
import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False


class SA(object):
    def __init__(self, num_city, data):
        self.T0 = 4000            # 退火的初始温度
        self.Tend = 1e-3          # 温度阈值，小于等于该阈值迭代就要停止
        self.rate = 0.9995        # 退火温度下降率
        self.num_city = num_city
        self.scores = []
        self.location = data      # 每个城市点的坐标
        self.fires = []
        self.dis_mat = self.compute_dis_mat(num_city, data)             # 求一个距离矩阵，dist[i][j] 是城市 i 到 城市 j 的距离
        self.fire = self.greedy_init(self.dis_mat, 100, self.num_city)  # 找到贪婪初始化下最优的一个解
        # 显示初始化后的路径
        init_pathlen = 1. / self.compute_pathlen(self.fire, self.dis_mat)
        init_best = self.location[self.fire]
        # 存储每个温度下的最终路径，画出收敛图
        self.iter_x = [0]
        self.iter_y = [1. / init_pathlen]

    def greedy_init(self, dis_mat, num_total, num_city):      # dis_mat:距离矩阵，num_total:一个迭代轮次要计算多少条路径， num_city: 城市数量
        start_index = 0
        result = []
        for i in range(num_total):
            rest = [x for x in range(0, num_city)]
            # 所有起点都已经生成了
            if start_index >= num_city:
                start_index = np.random.randint(0, num_city)
                result.append(result[start_index].copy())     # 初始时 start_index 从 0 增长到 num_city, 超过后随机选择一个 start_index
                continue
            current = start_index
            rest.remove(current)
            # 找到从 start_index 出发，贪心算法找到的最短路径
            result_one = [current]
            while len(rest) != 0:
                tmp_min = math.inf
                tmp_choose = -1
                for x in rest:
                    if dis_mat[current][x] < tmp_min:
                        tmp_min = dis_mat[current][x]
                        tmp_choose = x
                current = tmp_choose
                result_one.append(tmp_choose)
                rest.remove(tmp_choose)
            result.append(result_one)
            start_index += 1

        pathlens = self.compute_paths(result)  # 计算所有路径的距离
        sortindex = np.argsort(pathlens)       # 找到最小值所在的下标
        index = sortindex[0]
        return result[index]                   # 返回路径最短的那条路径

    # 初始化一条随机路径
    def random_init(self, num_city):
        tmp = [x for x in range(num_city)]
        random.shuffle(tmp)          # 打乱 tmp 数组
        return tmp

    # 计算不同城市之间的距离
    def compute_dis_mat(self, num_city, location):
        dis_mat = np.zeros((num_city, num_city))
        for i in range(num_city):
            for j in range(num_city):
                if i == j:
                    dis_mat[i][j] = np.inf
                    continue
                a = location[i]
                b = location[j]
                tmp = np.sqrt(sum([(x[0] - x[1])**2 for x in zip(a, b)]))
                dis_mat[i][j] = tmp
        return dis_mat

    # 计算路径长度
    def compute_pathlen(self, path, dis_mat):
        a = path[0]
        b = path[-1]
        result = dis_mat[a][b]
        for i in range(len(path) - 1):
            a = path[i]
            b = path[i+1]
            result += dis_mat[a][b]
        return result

    # 计算一个温度下产生的一个群体的长度
    def compute_paths(self, paths):
        result = []
        for one in paths:
            length = self.compute_pathlen(one, self.dis_mat)
            result.append(length)
        return result

    # 产生一个新解：随即交换两个元素的位置
    def get_new_fire(self, fire):
        fire = fire.copy()
        t = [x for x in range(len(fire))]    # t 是下标数组
        a, b = np.random.choice(t, 2)        # 随机选择两个值
        fire[a:b] = fire[a:b][::-1]          # a-b 之间倒置
        return fire

    # 退火策略，根据温度变化有一定概率接受差的解
    def eval_fire(self, raw, get, temp):
        len1 = self.compute_pathlen(raw, self.dis_mat)
        len2 = self.compute_pathlen(get, self.dis_mat)
        dc = len2 - len1
        p = max(1e-1, np.exp(-dc/temp))     # dc 解的差，temp 是温度
        if len2 < len1:
            return get, len2
        elif np.random.rand() <= p:
            return get, len2
        else:
            return raw, len1   # 即使有一定概率可以接受差的解，但是这里也没有接受到这个解

    # 模拟退火总流程
    def sa(self):
        count = 0
        # 记录最优解
        best_path = self.fire      # 只有一个最优解
        best_length = self.compute_pathlen(self.fire, self.dis_mat)

        while self.T0 > self.Tend:
            count += 1
            # 产生在这个温度下的随机解，在一个迭代轮次内应该搜索很多次，以保证搜索彻底
            tmp_new = self.get_new_fire(self.fire.copy())
            # 根据温度判断是否选择这个解
            self.fire, fire_len = self.eval_fire(best_path, tmp_new, self.T0)
            # 更新最优解
            if fire_len < best_length:
                best_length = fire_len
                best_path = self.fire
            # 降低温度
            self.T0 *= self.rate
            # 记录收敛曲线
            self.iter_x.append(count)
            self.iter_y.append(best_length)
            print(count, best_length)
        return best_length, best_path

    def run(self):
        best_length, best_path = self.sa()
        return self.location[best_path], best_length


# 读取数据
def read_tsp(path):
    lines = open(path, 'r').readlines()
    assert 'NODE_COORD_SECTION\n' in lines
    index = lines.index('NODE_COORD_SECTION\n')
    data = lines[index+1:-1]
    tmp = []
    for line in data:
        line = line.strip().split(' ')
        if line[0] == 'EOF':
            continue
        tmpline = []
        for x in line:
            if x == '':
                continue
            else:
                tmpline.append(float(x))
        if not tmpline:
            continue
        tmp.append(tmpline)
    data = tmp
    return data


if __name__ == "__main__":
    data = read_tsp("st70.tsp")
    data = np.array(data)
    data = data[:, 1:]
    show_data = np.vstack([data, data[0]])    # 要回到原点
    Best, Best_path = math.inf, None

    model = SA(num_city=data.shape[0], data=data.copy())
    path, path_len = model.run()
    print(path_len)

    if path_len < Best:
        Best = path_len
        Best_path = path
    # 加上起点
    Best_path = np.vstack([Best_path, Best_path[0]])

    # 画图
    fig, axs = plt.subplots(2, 1, sharex=False, sharey=False)
    axs[0].scatter(Best_path[:, 0], Best_path[:, 1])
    axs[0].plot(Best_path[:, 0], Best_path[:, 1])
    axs[0].set_title("规划结果")

    iterations = model.iter_x
    best_record = model.iter_y

    axs[1].plot(iterations, best_record)
    axs[1].set_title("收敛曲线")
    plt.show()
