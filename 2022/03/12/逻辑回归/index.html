йҖ»иҫ‘еӣһеҪ’

1. д»ҺзәҝжҖ§еӣһеҪ’еҲ°йҖ»иҫ‘еӣһеҪ’

вҖӢ зәҝжҖ§еӣһеҪ’жұӮзҡ„жҳҜиҫ“еҮәзү№еҫҒеҗ‘йҮҸ Y е’Ңиҫ“е…Ҙж ·жң¬зҹ©йҳө X д№Ӣй—ҙзҡ„зәҝжҖ§е…ізі» $\theta$ пјҢж»Ўи¶і $Y = X\theta$ гҖӮжӯӨж—¶жҲ‘д»¬зҡ„ Y жҳҜиҝһз»ӯзҡ„пјҢжүҖд»ҘжҳҜеӣһеҪ’жЁЎеһӢпјҢеҰӮжһңжҲ‘иҰҒжғіиҰҒ Y жҳҜзҰ»ж•ЈеҖјпјҢеҲҷеҸҜд»ҘеҶҚеҜ№ Y иҝӣиЎҢдёҖж¬ЎиҪ¬жҚўпјҢеҸҳдёә $g(Y)$пјҢеҰӮжһң $g(Y)$ зҡ„еҖјеңЁжҹҗдёӘеҢәй—ҙеҶ…жҳҜзұ»еҲ« AпјҢеңЁеҸҰдёҖдёӘеҢәй—ҙеҶ…жҳҜ BпјҢиҝҷж ·е°ұеҫ—еҲ°дәҶдёҖдёӘеҲҶзұ»жЁЎеһӢгҖӮеҰӮжһңзұ»еҲ«дёӘж•°дёә 2пјҢйӮЈд№Ҳе°ұжҳҜдёҖдёӘдәҢеҲҶзұ»жЁЎеһӢгҖӮ

2. дәҢе…ғйҖ»иҫ‘еӣһеҪ’жЁЎеһӢ

вҖӢ йҖ»иҫ‘еӣһеҪ’жЁЎеһӢе°ұжҳҜеңЁзәҝжҖ§жЁЎеһӢзҡ„еҹәзЎҖд№ӢдёҠеҠ дёҖдёӘеҮҪж•°иҪ¬жҚўдёәзҰ»ж•ЈеҖјпјҢиҝҷдёӘеҮҪж•°еңЁйҖ»иҫ‘еӣһеҪ’йҮҢйқўжҳҜ sigmod еҮҪж•°пјҡ$g(z) = \frac{1}{1+e^{-z}}$ пјҢдёәд»Җд№ҲжҳҜиҝҷдёӘеҮҪж•°пјҡ

  • еҪ“ $z$ и¶Ӣиҝ‘дәҺжӯЈж— з©·ж—¶пјҢ$g(z)$ и¶ӢдәҺ 1 пјҢиҖҢеҪ“ z и¶ӢдәҺиҙҹж— з©·ж—¶пјҢ$g(z)$ и¶ӢдәҺ0пјҢиҝҷдёӘжҖ§иҙЁеҫҲйҖӮеҗҲз”ЁжқҘиҝӣиЎҢеҲҶзұ»гҖӮ
  • sigmod еҮҪж•°иҝҳжңүдёҖдёӘеҫҲеҘҪзҡ„еҜјж•°жҖ§иҙЁпјҢ$g^{вҖҳ}(z) = g(z)(1-g(z))$

йҖ»иҫ‘еӣһеҪ’жЁЎеһӢпјҡ

вҖӢ $h_{\theta}(x) = \frac{1}{1+e^{-x\theta}}$

$x$ дёәж ·жң¬иҫ“е…ҘпјҢ$h_{\theta}(x)$ дёәжЁЎеһӢиҫ“еҮәпјҢеҸҜд»ҘзҗҶи§ЈдёәжҹҗдёҖзұ»еҲ«зҡ„жҰӮзҺҮпјҢиҖҢ $\theta$ дёәжЁЎеһӢеҸӮж•°пјҢеҜ№дәҺдәҢе…ғеҲҶзұ»пјҢеҰӮжһң $h_{\theta}(x) > 0.5$пјҢеҚі $x\theta > 0$пјҢйӮЈд№Ҳ y йў„жөӢдёә 1пјҢеҰӮжһң $h_{\theta}(x) < 0.5$ пјҢеҚі $x\theta < 0$пјҢйӮЈд№Ҳ y йў„жөӢдёә 0 гҖӮ

йҖ»иҫ‘еӣһеҪ’зҡ„зҹ©йҳөеҪўејҸпјҡ

$h_{\theta}(X) = \frac{1}{1+e^{-X\theta}}$пјҢ$h_{\theta}(X)$ жҳҜжЁЎеһӢиҫ“еҮәпјҢжҳҜ mx1 зҡ„з»ҙеәҰпјҢ$X$ жҳҜзү№еҫҒзҹ©йҳөпјҢжҳҜ mxn зҡ„з»ҙеәҰпјҢ$\theta$ жҳҜеҲҶзұ»жЁЎеһӢзҡ„еҫ…жұӮзі»ж•°пјҢжҳҜ nx1 зҡ„з»ҙеәҰгҖӮ

3. дәҢе…ғйҖ»иҫ‘еӣһеҪ’зҡ„жҚҹеӨұеҮҪж•°

вҖӢ зәҝжҖ§еӣһеҪ’зҡ„иҫ“еҮәжҳҜиҝһз»ӯеҖјпјҢжүҖд»ҘеҸҜд»ҘдҪҝз”Ёе№іж–№е·®жқҘе®ҡд№үжҚҹеӨұеҮҪж•°пјҢдҪҶжҳҜйҖ»иҫ‘еӣһеҪ’зҡ„иҫ“еҮәдёҚжҳҜиҝһз»ӯеҖјпјҢжүҖд»Ҙе№іж–№е·®жҚҹеӨұеҮҪж•°дёҚйҖӮеҗҲйҖ»иҫ‘еӣһеҪ’пјҢдёҚиҝҮеҸҜд»ҘдҪҝз”ЁжңҖеӨ§дјјз„¶жі•жҺЁеҜјеҮәжҚҹеӨұеҮҪж•°гҖӮ

вҖӢ еҜ№дәҺдәҢе…ғйҖ»иҫ‘еӣһеҪ’пјҢеҒҮи®ҫиҫ“еҮә y еҸӘжңү 0 е’Ң 1дёӨзұ»гҖӮ

вҖӢ $P(y=1|x,\theta) = h_{\theta}(x)$

вҖӢ $P(y=0|x,\theta) = 1-h(\theta)$

еҶҷжҲҗдёҖдёӘеҮҪж•°е°ұжҳҜпјҡ$P(y|x,\theta) = h_{\theta}(x)^{y}(1-h_{\theta}(x))^{1-y}$ пјҢy еҸӘиғҪеҸ– 0 е’Ң 1 гҖӮ

йҖҡиҝҮдјјз„¶еҮҪж•°жңҖеӨ§еҢ–жқҘжұӮи§ЈеҸӮж•° $\theta$ пјҢиҝӣиЎҢиҪ¬жҚўпјҢе°Ҷдјјз„¶еҮҪж•°еҸ–еҜ№ж•°е№¶еҸ–еҸҚдҪңдёәжҚҹеӨұеҮҪж•°пјҢз„¶еҗҺжңҖе°ҸеҢ–жҚҹеӨұеҮҪж•°гҖӮ

дјјз„¶еҮҪж•°пјҡ$L(\theta) = \prod_{i=1}^{m}(h_{\theta}(x^{(i)}))^{y^{(i)}}(1-h_{\theta}(x^{(i)}))^{1-y^{(i)}}$

жҚҹеӨұеҮҪж•°пјҡ$J(\theta) = -lnL(\theta) = -\sum_{i=1}^{m}(y^{(i)}log(h_{\theta}(x^{(i)}))+(1-y^{(i)})log(1-h_{\theta}(x^{(i)})))$

жҚҹеӨұеҮҪж•°зҡ„зҹ©йҳөиЎЁејҸпјҡ$J(\theta) = -Y^Tlogh_{\theta}(X) - (E-Y)^Tlog(E-h_{\theta}(X))$пјҢе…¶дёӯ E дёәе…Ё 1 еҗ‘йҮҸгҖӮ

4. дәҢе…ғйҖ»иҫ‘еӣһеҪ’жҚҹеӨұеҮҪж•°зҡ„дјҳеҢ–ж–№жі•

вҖӢ иҝҷйҮҢеҸӘд»Ӣз»ҚжўҜеәҰдёӢйҷҚжі•

$\frac{\partial}{\partial\theta} = X^T[\frac{1}{h_{\theta}(X)}\odot h_{\theta}(X)\odot(E-h_{\theta}(X))\odot(-Y)] + X^T[\frac{1}{E-h_{\theta}(X)}\odot h_{\theta}(X)\odot(E-h_{\theta}(X))\odot(E-Y)]$

з®ҖеҢ–дёәпјҡ

$\frac{\partial}{\partial\theta}J(\theta) = X^T(h_{\theta}(X)-Y)$

иҝӯд»Је…¬ејҸдёәпјҡ

$\theta = \theta - \alpha X^{T}(h_{\theta}(X) - Y)$ , $\alpha$ дёәжӯҘй•ҝ гҖӮ

5. дәҢе…ғйҖ»иҫ‘еӣһеҪ’зҡ„жӯЈеҲҷеҢ–

вҖӢ йҖ»иҫ‘еӣһеҪ’д№ҹдјҡжңүиҝҮжӢҹеҗҲзҡ„й—®йўҳпјҢд№ҹйңҖиҰҒиҖғиҷ‘жӯЈеҲҷеҢ–пјҢеёёи§Ғзҡ„жңү $L1$ жӯЈеҲҷеҢ– е’Ң $L2$ жӯЈеҲҷеҢ–гҖӮ

$L1$жӯЈеҲҷеҢ–

$J(\theta) = -Y^Tlogh_{\theta}(X) - (E-Y)^Tlog(E-h_{\theta}(X)) + \alpha\left | \theta \right |_{1}$

$L2$ жӯЈеҲҷеҢ–

$J(\theta) = -Y^Tlogh_{\theta}(X) - (E-Y)^Tlog(E-h_{\theta}(X)) + \frac{1}{2}\alpha\left | \theta \right |_{2}^{2}$

6. еӨҡе…ғйҖ»иҫ‘еӣһеҪ’

еҜ№дәҺдәҢе…ғйҖ»иҫ‘еӣһеҪ’пјҡ

вҖӢ $P(y=1|x,\theta) = h_{\theta}(x) = \frac{1}{1+e^{-x\theta}} = \frac{e^{x\theta}}{1+e^{x\theta}}$

вҖӢ $P(y=0|x,\theta) = 1-h(\theta) = \frac{1}{1+e^{x\theta}}$

вҖӢ дёӨеҲҷзӣёйҷӨеҸ–еҜ№ж•°

вҖӢ $ln\frac{P(y=1|x,\theta) }{P(y=0|x,\theta)} = x\theta$
вҖӢ

жҺЁе№ҝеҲ°еӨҡе…ғпјҢеҒҮи®ҫ y зҡ„еҖјеҸҜд»ҘеҸ–пјҡ$1,2,3\dots K$

$ln\frac{P(y=1|x,\theta) }{P(y=K|x,\theta)} = x\theta_{1}$

$ln\frac{P(y=2|x,\theta) }{P(y=K|x,\theta)} = x\theta_{2}$

$\dots$

$ln\frac{P(y=K-1|x,\theta) }{P(y=K|x,\theta)} = x\theta_{K-1}$

$\sum_{i=1}^{K}P(y=i|x,\theta) = 1$

дёҠйқўжңү K дёӘж–№зЁӢпјҢжұӮи§ЈиҜҘж–№зЁӢеҫ—пјҡ

$P(y=k|x,\theta) = \frac{e^{x\theta_{k}}}{1+\sum_{t=1}^{K-1}e^{x\theta_{t}}}, k = 1,2,\dots K-1$

$P(y=k|x,\theta) = \frac{1}{1+\sum_{t=1}^{K-1}e^{x\theta_{t}}}$

еӨҡе…ғйҖ»иҫ‘еӣһеҪ’зҡ„жҚҹеӨұеҮҪж•°е’ҢдјҳеҢ–ж–№жі•е’ҢдәҢе…ғйҖ»иҫ‘еӣһеҪ’зұ»дјј гҖӮ
